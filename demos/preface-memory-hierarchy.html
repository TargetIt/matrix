GMEM

容量大 / 延迟高 / 适合存放完整矩阵

SMEM

线程块共享缓存 / 适合 block tile

Registers

线程私有、最快 / 适合累加器与 thread tile

当 reuse 很低时，你几乎每做几次 FMA 就要重新回到 GMEM 取数据；当 reuse 提高后，GMEM 读取被更多片上消费摊薄，吞吐才有机会逼近计算峰值。

建议观察：红色粒子代表从 GMEM 搬运的 A/B tile，蓝色粒子代表 SMEM 中的共享数据，绿色粒子代表进入寄存器后的私有片段。reuse 越高，片上消费次数越多。
