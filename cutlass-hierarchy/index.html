主循环流水

双缓冲开启时，三条指令流重叠。

GMEM → SMEM加载下一个 A/B tile
SMEM → RFwarp 取下一组 fragment
RF → Math当前 outer product
层级与存储一致

Block tile 对应 shared memory，warp/thread accumulator 对应 register file；越靠近计算单元，容量越小、复用越密集。

更大 tile 并非总是更快

tile 增大提升数据复用，也增加 SMEM 与寄存器压力，可能减少一个 SM 上的并发 block 数。

Epilogue 是融合入口

CUTLASS 可将 bias、ReLU 等逐元素操作放进 epilogue，避免额外 kernel launch 和一次 GMEM 往返。
